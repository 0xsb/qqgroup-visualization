# qqgroup-visualization

> QQ群关系数据客查询（表格版）

## 使用方法
```
npm install
1.安装SqlServer2008R2
2.下载数据库备份文件
3.恢复数据库到SqlServer
4.修改db.json，配置自己的数据库连接信息
5.npm run start
```
