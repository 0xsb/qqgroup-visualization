# qqgroup-visualization

> QQ群关系数据客查询（表格版）

## 使用方法
因为数据库是SqlServer2008R2，所以推荐使用Windows7以上操作系统
+ 下载安装SqlServer2008R2
+ 下载数据库备份，恢复数据库到SqlServer2008R2
+ 下载安装Node.js环境
+ 配置db.json文件，修改其中数据库连接为自己的数据库
+ npm install & npm run start
