# qqgroup-visualization

> QQ群关系数据客查询（表格版）

## 使用方法
因为数据库是SqlServer2008R2，所以推荐使用Windows7以上操作系统
+ 下载安装SqlServer2008R2
+ 下载[数据库备份](http://www.baidu.com/)，恢复数据库到SqlServer2008R2
+ 下载安装Node.js环境
+ 配置项目根目录下的db.json文件，修改其中数据库连接信息为自己的数据库连接信息
+ npm install & npm run start
