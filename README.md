# qqgroup-visualization

> QQ群关系数据客查询（表格版）

## 使用方法
因为数据库是SqlServer2008，所以推荐使用Windows7以上操作系统
+ 下载安装SqlServer2008
+ 下载数据库备份，恢复数据库到SqlServer2008
